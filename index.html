Instructions (Manual)
This is for if u dont wanna get a mod launcher
1) Install Minecraft
2) Download and run Fabric installer from the official Fabric launcher.
3) Set Minecraft Version and Fabric Version to the versions listed above. Leave everything else.
4) A new profile should be created in your minecraft launcher
5) Download allmods.zip from the drive link below
6) Type in %AppData%/.minecraft in the file explorer bar
7) If there is no mods folder, run the fabric mod profile and close it when you get to title screen
8) Move the mod .jar files from the bbb24mods.zip into the mods folder
9) Run modded profile
